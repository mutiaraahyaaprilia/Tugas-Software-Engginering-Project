# Sistem Absensi Mahasiswa Berbasis GPS

Proyek ini adalah sistem absensi mahasiswa menggunakan teknologi Geofencing untuk memastikan mahasiswa berada di radius kampus saat melakukan absensi melalui perangkat mobile.

## Isi Repositori
1. **analisis_design_implementasi.txt**: Dokumen lengkap yang berisi analisis masalah, perancangan sistem (use case & UI), serta rencana implementasi teknis.

## Fitur Utama
- Deteksi lokasi real-time dengan GPS.
- Pembatasan radius absensi (Geofencing).
- Dashboard pemantauan kehadiran untuk dosen.
- Manajemen data mahasiswa dan jadwal untuk admin.
